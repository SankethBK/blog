# rabin2 — Binary Inspection Tool (Radare2 Suite)

`rabin2` is the **binary analysis frontend** of the Radare2 toolkit.  
It extracts metadata, headers, imports, exports, symbols, and sections from executables (ELF, PE, Mach-O, etc.) **without loading or executing** them.  
Think of it as a deeper and faster combination of `readelf`, `objdump`, and `strings`.

---

## ⚙️ 1. Basic Usage

```bash
rabin2 -I binary        # General information
rabin2 -S binary        # Section table
rabin2 -s binary        # Symbols
rabin2 -i binary        # Imports
rabin2 -x binary        # Extract all strings
rabin2 -z binary        # Show strings from .rodata and data
rabin2 -r binary        # Relocations
rabin2 -l binary        # Linked libraries
rabin2 -e binary        # Entrypoint
rabin2 -m binary        # Show main address (if found)
```

---

## 2. General Info

```bash
rabin2 -I matryoshka
```

**Output example:**
```
arch     x86
bits     64
os       linux
endian   little
class    ELF64
type     DYN (Position Independent Executable)
machine  AMD x86-64
entry    0x10d0
baddr    0x0
canary   true
nx       true
pie      true
relro    partial
```

> `rabin2 -I` is your go-to for a quick “what am I looking at?” summary.  
> It pulls together what `file`, `readelf -h`, and `checksec` would show.

---

## 3. Sections and Segments

```bash
rabin2 -S matryoshka
```

Shows `.text`, `.data`, `.rodata`, `.plt`, `.got`, etc.

| Column | Meaning |
|---------|----------|
| name | Section name |
| size | Size of section |
| vaddr | Virtual address (runtime) |
| paddr | Physical offset (in file) |
| perm | Permissions (r/w/x) |

You can filter:
```bash
rabin2 -S | grep text
```

---

## 4. Imports, Exports & Symbols

### Imports (external functions)
```bash
rabin2 -i binary
```
Shows dynamically linked libc symbols like:
```
puts
execve
fwrite
memfd_create
```

### Exports (functions defined in binary)
```bash
rabin2 -E binary
```

### All symbols (global + local)
```bash
rabin2 -s binary
```

---

##  5. Strings & Data

```bash
rabin2 -z binary
```
Shows printable ASCII and UTF-16 strings.  
Use with `-zz` to scan all sections instead of only `.rodata`.

Example:
```
vaddr=0x00104080 paddr=0x00004080 len=20 section=.rodata string=usage: ./matryoshka key
```

Or dump all into a file:
```bash
rabin2 -zzq binary > strings.txt
```

---

## 6. Relocations and Libraries

```bash
rabin2 -r binary
```
Shows `.rela.dyn` and `.rela.plt` relocation entries, similar to:
```
000000004000  R_X86_64_JUMP_SLO  memfd_create
000000004008  R_X86_64_JUMP_SLO  puts
000000004010  R_X86_64_JUMP_SLO  execve
```

Linked libraries:
```bash
rabin2 -l binary
# => libc.so.6, ld-linux-x86-64.so.2
```

---

## 7. Entrypoint and Addresses

```bash
rabin2 -e binary     # Show entrypoint address
rabin2 -m binary     # Guess address of main()
```

---

## 8. Extract Raw Sections

You can dump raw data from sections or symbols.

```bash
rabin2 -O .rodata binary > rodata.bin
rabin2 -O .text binary > text.bin
```

---

## 9. Quick Recap Table

| Command | Purpose | Similar to |
|----------|----------|-------------|
| `rabin2 -I` | General info (arch, PIE, NX, canary, etc.) | `file + checksec` |
| `rabin2 -S` | Section headers | `readelf -S` |
| `rabin2 -s` | Symbols | `nm` |
| `rabin2 -i` | Imports | `readelf -s` |
| `rabin2 -E` | Exports | `nm --defined-only` |
| `rabin2 -r` | Relocations | `readelf -r` |
| `rabin2 -l` | Linked libs | `ldd` |
| `rabin2 -z` | Strings | `strings` |
| `rabin2 -O` | Dump section | `objcopy` |
| `rabin2 -m` | Find main | heuristic |
| `rabin2 -e` | Entrypoint | `readelf -h` |

---

## 10. Combine with r2 (radare2)

When deeper analysis is needed:
```bash
r2 -A binary
```
`rabin2` ≈ quick metadata dump,  
`r2` ≈ full disassembly + decompilation workspace.
